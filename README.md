# CS4300 - Homework 1

This assignment focuses on creating projects in Python and testing the code. 

## Requirements

- Python 3.7+  
- NumPy  
- Pytest  

## How to Run the Code

To run a task file, use the following command:
`python task1.py`


## How to Run the Tests

To run all tests, use:
`pytest`
To run a specific test file:
`pytest test_task1.py`
